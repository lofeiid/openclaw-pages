split

方便面 最容易讲散的地方

信息太多

当素材很多时，更应该先做取舍，不要把每个事实都塞进同一层级。

  • 删掉和主线无关的枝节
  • 一页只保留一个判断
  • 让观众每翻一页都有推进感
结构太弱

信息本身没问题，但如果没有章节感和对比感，观众会觉得像连续长文。

  • 用 divider 建立章节感
  • 用 table 和 dashboard 创造结构差异
  • 用 quote 或 full-bleed 做节奏变化
dashboard

把 方便面 讲清楚的 4 个设计信号

这是结构和视觉需要一起达到的基本线。

Hierarchy 强

标题、副标题、正文一眼可分

Spacing 松

留白足够，重点不会互相打架

Rhythm 有变化

内容页、强调页、总结页节奏分明

Purpose 明确

每页都知道自己在推动什么结论
